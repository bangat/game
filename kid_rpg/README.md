# kid_rpg

모바일 가로모드 중심의 한국어 액션 RPG 초안입니다.

## 실행

```powershell
cd kid_rpg
npm install
npm start
```

브라우저에서 `http://localhost:3187` 또는 같은 네트워크의 `http://내PC아이피:3187`로 접속합니다.

## 현재 포함된 것

- 시작 화면과 직업 선택
- 전사 / 마법사 / 궁수
- Three.js 기반 2.5D 스타일 장면
- 기본 공격과 클래스별 스킬 2개
- 자동사냥과 자동 퀘스트 추적
- 인벤토리 / 장비 / 무기 강화
- 전투력, 레벨, 골드, 드랍
- 오프라인 보상
- 옵션의 배경음악 / 효과음 토글
- WebSocket 기반 온라인 인원 동기화
