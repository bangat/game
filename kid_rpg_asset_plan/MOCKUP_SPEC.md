# 잠룡/어나더던전급 Phaser ARPG 목업 명세

이 문서는 실제 화면, 씬 흐름, HUD 배치, 전투 연출, 대화 동선을 고정하기 위한 목업 기준이다.

## 1. 공통 화면 원칙

- 기본 화면은 `가로모드`.
- 세로모드는 `부분 지원`.
- 중앙 전투면은 항상 최대한 비워 둔다.
- HUD는 작고 얇게 유지한다.
- 버튼은 모서리로 밀고, 중앙에는 큰 패널을 띄우지 않는다.

## 2. 로딩씬

- 중앙: 로고/타이틀
- 하단: 진행 문구
- 배경: `객잔`, `필드`, `던전`별 교체 가능한 일러스트 슬롯
- 보조: 짧은 팁 문구 영역
- 목적: 맵 전환의 이질감을 줄이고 다음 지역 분위기를 미리 보여준다.

## 3. 객잔

- 상단 얇은 상태바
- 우측 세로 메뉴 최소 버튼
- 좌측 또는 중앙에 메인 NPC 배치
- 하단 미니 퀘스트바
- 객잔 UI 클릭 시 `필드` 또는 `던전`으로 이동
- 대화, 상점, 우편, 설정은 `슬림 패널`
- 객잔은 특정 장르를 따라 하기보다 `시작 허브`, `상점`, `퀘스트`, `던전 진입`이 모이는 compact hub 역할을 한다.

## 4. 초보 필드

- 소형 HUD + 넓은 전투면
- 자동사냥 가능
- 필드 몬스터 다수 스폰
- 몬스터 위에 `이름`, `체력`, `데미지 텍스트`
- 스킬 드랍/아이템 드랍 표시
- 초보 필드는 첫 성장 루프를 보여주는 공간이다.

## 5. 던전

- 필드보다 어두운 톤
- 입장 직전에 짧은 로딩씬 재생
- 던전 몬스터/보스는 별도 스폰 테이블 사용
- 던전은 구조가 더 조밀하고 위험도가 높아 보여야 한다.

## 6. 메인 퀘스트 연출

- 퀘스트 버튼 클릭 시 하단 말풍선형 대화 진행
- 대화는 짧고 빠르게 읽히는 모바일 ARPG식 로그 형식으로 간다.
- NPC를 직접 누르지 않아도 메인 퀘스트 흐름이 이어진다.
- 대화 완료 후 퀘스트 목표가 자동 갱신된다.

## 7. 전투 HUD

- 상단: 캐릭터 정보, 체력/마나/레벨, 전투력, 재화
- 우측 세로 메뉴: 가방, 퀘스트, 우편, 설정, 상점
- 좌하단: 조이스틱
- 우하단:
- `기본 공격 1`
- `스킬 3칸`
- `AUTO 1칸`
- `버프/자동물약 2칸`
- 하단: 미니 퀘스트바 및 진행 상태

## 8. 전투 연출

- 기본 공격 1종
- 액티브 스킬 3칸
- AUTO 버튼
- 버프/자동물약 2칸
- 히트 스톱
- 피격 플래시
- 카메라 쉐이크
- 크리티컬 텍스트
- 몬스터 위 데미지 숫자
- 드랍 이펙트는 가볍지만 눈에 띄어야 한다.

## 9. 세로모드 부분 지원

- 세로에서는 `객잔`, `퀘스트`, `인벤토리`, `스토리`, `검수 확인`, `메뉴 확인` 위주로 대응한다.
- 전투는 축소형 HUD로 최소 대응하거나 `가로 전환 권장` 안내를 띄운다.
- 세로에서도 화면이 깨지거나 조작이 불가능해 보이면 안 된다.

## 10. 미래 확장 전제

- 캐시 상점 패널 확장 가능
- 신규 스킬 슬롯/페이지 확장 가능
- 신규 이벤트 배너/이벤트 던전 슬롯 확장 가능
- 필드 보스/던전 보스 타이머 UI 확장 가능
- 상점, 우편, 시즌, 이벤트, PvP 탭이 나중에 붙어도 레이아웃이 무너지지 않아야 한다.

## 11. 성능 및 옵션 연동

- 절전모드와 저프레임 모드가 즉시 반영되어야 한다.
- 프레임 옵션 변경 시 게임 루프와 이펙트 밀도 모두 조정된다.
- 데미지 텍스트, 화면 흔들림, 파티클 밀도는 설정과 연동되어야 한다.
- 몬스터가 20~50마리 보이는 상황에서도 옵션 단계별로 프레임 하락을 제어할 수 있어야 한다.
