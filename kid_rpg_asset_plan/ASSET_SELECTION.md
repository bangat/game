# 에셋 1차 선택 메모

## 목표

기존 `허접한 HTML UI + 뜨는 바닥 + 3D 잔상`을 버리고,
`어나더던전`처럼 도트 기반 모바일 ARPG 느낌으로 전면 교체한다.

## 레퍼런스 기준

- `Another Dungeon 공식 메인 플레이 화면`
  - 전투 영역 비율
  - 상단 상태창
  - 하단 스킬바
- `Another Dungeon 공식 다크 던전 무드`
  - 배경 명암
  - 붉은 포털 / 어두운 챕터 톤
- `Another Dungeon 공식 캐릭터 라인업`
  - 직업 초상 톤
  - 플레이어 본체 픽셀 밀도

## 버릴 방향

- 큼직한 HTML 박스형 UI
- 원근감 강한 3D 카메라
- 바닥과 오브젝트가 따로 노는 떠 있는 맵
- 저해상도 벡터형 캐릭터
- 통일감 없는 임시 프리미티브
